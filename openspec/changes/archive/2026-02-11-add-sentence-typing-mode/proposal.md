# Change: Add Sentence Typing Mode

## Why
为游戏添加新的句子打字模式，与经典的单词消除模式相辅相成。句子打字模式更贴近真实的打字场景，帮助用户在完整语境中练习打字，提升连续输入的流畅度和对标点符号、空格的处理能力。这是一种更具挑战性和实用性的练习方式。

## What Changes
- **游戏模式系统**: 引入游戏模式抽象，支持经典模式(Classic Mode)和句子模式(Sentence Mode)的切换
- **句子数据加载**: 从 `data/sentences.txt` 文件中随机读取一行句子作为游戏内容
- **句子输入处理**:
  - 支持字母、数字、标点符号、空格的输入
  - 实时对比用户输入和目标句子
  - 绿色显示正确字符，红色显示错误字符，一一对应显示
  - 支持 backspace 删除字符
- **完成判定**: 当用户输入长度达到句子长度时按回车键结束游戏（允许有错误）
- **UI 渲染**:
  - 欢迎页面增加模式选择菜单（在 start 之后进入模式选择）
  - 游戏界面显示目标句子和用户输入的对比
  - 统计数据适配句子模式（总字符数、正确字符数、错误字符数等）

这是全新功能，不涉及破坏性变更。

## Impact
- **新增能力规范**:
  - `game-modes` - 游戏模式抽象和切换机制

- **修改能力规范**:
  - `ui-rendering` - 新增模式选择菜单和句子模式渲染
  - `input-handling` - 扩展输入处理以支持句子模式

- **受影响代码**:
  - `pkg/game/game.go` - 需要支持句子模式的游戏逻辑
  - `pkg/ui/styles.go` - 需要添加句子模式的 UI 渲染
  - `cmd/word-killer/main.go` - 需要添加模式选择逻辑
  - `pkg/config/config.go` - 可能需要添加句子文件路径配置

- **外部依赖**:
  - 句子数据文件 `data/sentences.txt`（需要手动创建或提供示例）
