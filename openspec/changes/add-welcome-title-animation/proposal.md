# Change: Add Welcome Title Animation

## Why
为欢迎界面的 "Word Killer" 标题添加动态视觉效果，增强游戏启动时的沉浸感和趣味性。"Word" 的发抖效果传达单词的不稳定状态，"Killer" 的红色呼吸脉冲强调"杀手"主题，营造紧张刺激的氛围。

## What Changes
- **欢迎界面标题动画**: 为 "Word Killer" 添加持续循环的动画效果
  - "Word" 文字随机抖动效果（每个字符独立随机偏移）
  - "Killer" 文字红色呼吸脉冲渐变（从暗红到亮红循环）
- **动画帧管理**: 在 UI 渲染层添加动画帧计数器，支持持续循环动画
- **不破坏现有功能**: 保持所有现有 UI 样式和功能不变

这是新增功能，不涉及破坏性变更。

## Impact
- **新增能力规范**:
  - `ui-rendering` - 添加欢迎标题动画渲染能力

- **受影响代码**:
  - `pkg/ui/styles.go` - 需要修改 `RenderWelcome()` 函数，添加动画效果
  - 可能需要添加新的动画辅助函数

- **外部依赖**:
  - 无新增依赖，使用现有的 lipgloss 库
